Example of other parameters.

hasNullIcon - Indicate if the selector allow Null Selection, default to false.

currSelection - Indicate the initial selection index of the IconPicker, default to 0.

currSelectionUrl - Indicate the initial display image of the IconPicker. If the image url is existed in the icon selector, the currSelection will change to the specific index.

containerWidth and containerHeight - Indicate the width and height of the icon selector, default to 5 and 3.
